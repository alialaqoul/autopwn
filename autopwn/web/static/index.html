New tool

A tool runs an installed command line. It is built as an argv list (never a shell string): binary → subcommand → <positional> → [flags] → fixed. Parameters are derived from the positional and flag variables.

Name: a unique slug (no spaces), e.g. my_smb_check · Binary: the command as installed on PATH, e.g. nxc · Category groups it in the list.
Variables are canonical names Autopwn shares between tools (a value learned once auto-fills everywhere). Use: target · url · domain · username · password · nthash · userlist · passlist · wordlist · hashfile · dc_ip · base_dn.
Captured variables are added to Engagement Variables and auto-filled into other tools that take a parameter of the same name.
